Global compilation

Here the full global compilation of glacial 10Be and 26Al data is presented in tab-delimited text files. The file name shows the timing of the dataset with expage-201606.txt presenting data from year 2016 and month 06.

 

Global compilation kml: expage.kml (from the most recent dataset)

Global compilation log: expage-log.txt



The exposure age data is presented in the following text files:

expage – Tab-delimited data file with 10Be and 26Al exposure age data.

expage-comments – Comment file describing data sources other than the primary publication and data assumptions.

expage-refs – List of references.



To display special characters in the text files correctly, use UTF-8 (unicode) encoding.
